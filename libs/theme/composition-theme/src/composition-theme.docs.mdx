This modulee provides three built-in theme variants out of the box.

@compendium/theme.default-theme
: The default theme will adapt itself to match the system prefered color scheme

@compendium/theme.dark-theme
: Applies a dark theme regardless of user preferences

@compendium/theme.light-theme
: Applies a light theme regardless of user preferences
